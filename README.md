# Connector

This tool can be used to interact with EVM (Ethereum Virtual Machine) based blockchains.

## Technical stuff

### Technologies used

* Node v14.15
* Solidity 0.7.5
* Hardhat
* Ethers.js
* ReactJS
* TypeScript
* ESLint / Prettier

### Instructions

Please read INSTRUCTIONS.md and scripts in package.json

```
npm run localchain
```

```
npm run deploy-local
```

```
npm run serve-frontend
```
